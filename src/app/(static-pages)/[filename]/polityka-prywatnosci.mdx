# Polityka prywatnosci

Oto polityka prywatnosci.

1. Nie zbieramy zadnych danych osobowych.
2. Nie mamy dostepu do danych osobowych.
3. Nie udostepniamy danych osobowych.
4. Nie sprzedajemy danych osobowych.
5. Nie przetwarzamy danych osobowych.
6. Nie przechowujemy danych osobowych.
7. Nie przekazujemy danych osobowych.
8. Nie udostepniamy danych osobom trzecim.
9. Nie udostepniamy danych organom panstwowym.
10. Nie udostepniamy danych organom prywatnym.
